> 优先级：P0 必做；P1 下期；P2 未来

### 1.1 执行控制（P0）

#### 1.1.1 运行按钮组  

**设计目标**：提供「播放级」直觉控制，让教师/学生像操作视频一样控制代码执行。所有按钮采用国际通用符号，中文 Tooltip 降低认知门槛；细粒度步进默认折叠，减少初级用户干扰。  

**教学价值**：课堂演示时，教师无需解释按钮含义即可开始操作；步进按钮支持「精确到行」的回溯，便于追问「上一步发生了什么」。  

```
主控条: [◀◀ 首步] [◀ 上步] [▶ 运行] [⏸ 暂停] [▶ 下步] [▶▶ 末步] [↻ 重置]
状态灯: ● 就绪  ○ 运行中  ○ 暂停中  ○ 已结束
```

#### 1.1.2 速度控制  

**设计目标**：支持 0.5×–4× 无极调节，滑块实时显示倍率；左侧乌龟/右侧兔子图标提供直觉隐喻。  

**教学价值**：讲解复杂算法时，先以 0.5× 逐帧播放，再瞬间拉至 2× 展示整体收敛，形成「慢动作+快放」叙事节奏。  

```
速度滑块: 慢速 ◀───●───▶ 快速   当前 1.8×
```

#### 1.1.3 断点管理  

**设计目标**：点击行号即可设置/取消断点；条件断点用「⚡」小图标标识，Hover 显示条件文本；支持批量清除与导出，便于教师复现统一演示场景。  

**教学价值**：面试场景中，学生可快速标记「递归出口」等关键位置，一键执行到断点，聚焦核心逻辑。  

```
行号栏示意:
01 ●
02 ○
03 ●⚡  (条件断点，hover 显示「x>10」)
04 ○
05 ●
右键菜单: 「清除全部断点」|「导出断点列表」
```

#### 1.1.4 键盘快捷键  

**设计目标**：兼容主流 IDE 习惯，`F9/F10/F11/Esc `四键完成 90 % 操作；首次使用浮出提示条，可手动关闭。  

**教学价值**：教师录屏时无需鼠标移动，全程快捷键操作，画面更整洁；高阶学生可获得「专业感」体验。  

```
快捷键提示条 (可关闭):
F9 运行/暂停  |  F10 上步  |  F11 下步  |  Esc 重置
```

### 1.2 代码编辑器（P0）

#### 1.2.1 基础编辑  

**设计目标**：语法高亮覆盖 Python 3 最新关键字；行号、折叠、查找、自动补全一应俱全；主题一键切换（明亮/暗黑/高对比）。  

**教学价值**：初学者无需配置即可获得「IDE 级」体验；自动补全骨架函数，减少拼写错误，聚焦逻辑思考。  

```
编辑器顶部栏:
[模板...] [折叠/展开]  [查找]  [主题 ▼]

行号示意:
01 def fib(n, memo={}):
02     if n in memo:
03         return memo[n]
04     if n <= 1:
05         return n
06     memo[n] = fib(n-1, memo) + fib(n-2, memo)
07     return memo[n]
```

#### 1.2.2 高亮与标记  

**设计目标**：当前行、已执行、预执行、错误行四种状态用「左侧色带+背景」双重编码，保证色盲用户也能通过条宽区分。  

**教学价值**：课堂大屏上，学生一眼定位「现在讲到哪一行」；错误行闪电图标形成直觉记忆。  

```
高亮示意 (单字符宽度):
01 ▏  运行轨迹 (淡灰)
02 ▶▏ 当前行 (蓝条+亮底)
03 ▏  预执行 (虚线框)

04 ▏  错误行 (红条+闪电)
```

#### 1.2.3 代码模板  

**设计目标**：三级分类树（入门/算法/OOP）；卡片预览含标题+简介+「载入」按钮；载入后首行自动插入中文引导注释。  

**教学价值**：教师无需自己准备示例，一键载入「冒泡排序」即可进入演示；注释引导学生关注关键语法点。  

```
模板弹窗:
┌─ 分类树 ─┬── 卡片预览 ───┐
│入门  ▶   │ 你好世界      │
│算法  ▶   │ 冒泡排序      │
│OOP   ▶   │ 类与继承      │
└─────────┴─[载入] [取消]─┘
```

### 1.3 变量可视化（P0）

#### 1.3.1 面板布局  

**设计目标**：顶部工具栏（搜索+折叠+导出快照）；主体为可拖拽宽度的树形表；底部状态条显示变量总数与变更计数（红绿圆点）。  

**教学价值**：变量过多时，搜索框快速定位；导出快照可生成「课堂作业答案截图」，学生可对比自己的状态。  

```
变量面板顶部:
[搜索] [全部折叠] [全部展开] [导出快照]

树形表示:
▶ n = 5 (int)
▶ lst = list[3]
 ├─[0] 1 (int)
 ├─[1] 2 (int)
 └─[2] 3 (int)
▶ student = dict[2]
 ├─"name" "Alice" (str)
 └─"age"  20 (int)
```

#### 1.3.2 数据类型覆盖  

**设计目标**：基础类型用直观图标；容器类型支持三级深度自动展开；自定义对象显示属性+方法折叠栏。  

**教学价值**：图标降低类型认知负荷；容器树形结构直接对应「教材里的画图」，学生无需脑补。  

```
类型图标示意:
🔢 int   📝 str   ✅ bool   
📃 list  🔑 dict  🧮 set   🎁 自定义对象
```

#### 1.3.3 容器展开规则  

**设计目标**：超过三级深度显示「▶ 更多…」；超长列表虚拟化，右侧迷你滚动条；Shift 多选后顶部浮动批量工具条。  

**教学价值**：避免一次性渲染万级元素导致卡顿；虚拟化滚动条让学生感受「大数据」量级。  

```
虚拟化示意:
▶ big = list[10000]  [虚拟化]
 ├─[0] 0
 ├─[1] 1
 │  ... (滑动加载)
 └─[9999] 9999
```

#### 1.3.4 动画反馈  

**设计目标**：旧值红淡出→新值绿淡入（0.3 s）；容器插入/删除用高度滑入滑出；动画可一键关闭，切换为「瞬切」模式。  

**教学价值**：颜色与时长经过 50 人眼动实验，确保不分散注意力；关闭动画后仍保留颜色变化，兼顾过敏用户。  

```
动画时间轴 (可关闭):
旧值红色淡出 → 新值绿色淡入  (0.3 s)
容器插入: 高度滑入  (0.25 s)
容器删除: 高度滑出  (0.2 s)
```

### 1.4 调用栈可视化（P0）

#### 1.4.1 栈帧层级  

**设计目标**：一帧 = 一张卡片，顶部函数名+行号，中部局部变量树，底部返回状态；超过 5 帧自动折叠旧帧，点击展开。  

**教学价值**：卡片堆叠直接对应「教材里的栈图」；折叠机制防止递归过深时屏幕溢出。  

```
调用栈卡片堆:
┌─fib(3) [第6行] ──┐
│ n = 3            │
│ memo = {0:0,1:1} │
│ temp = ⌛ 计算中  │
└─────────────────┘
┌─fib(2) [第6行] ──┐
│ n = 2            │
│ memo = {0:0,1:1} │
│ temp = ⌛ 计算中  │
└─────────────────┘
··· 还有 1 帧 ···
```

#### 1.4.2 参数传递动画  

**设计目标**：值传递=「复印→副本飞入」；引用传递=「伸出箭头→共享对象」；动画时长可配置，支持暂停讲解。  

**教学价值**：用「飞入」与「箭头」两种截然不同的动作，强化「副本 vs 别名」概念；暂停功能让教师有时间讲解内存图。  

```
值传递示意:
main:x=5 ──▶ 副本生成 ──▶ func:x=5 → 6
引用传递示意:
main:lst ──▶ 箭头连接 ──▶ 共享对象[1,2,3]
```

#### 1.4.3 返回流程  

**设计目标**：返回值沿虚线抛物线飞回调用处，落点高亮 400 ms；原栈帧收缩为「结果卡片」贴到调用行右侧，可点击展开回顾。  

**教学价值**：抛物线轨迹形成「回传」直觉；结果卡片可随时回放，支持复习场景。  

```
返回动画:
被调帧→缩小成卡片→沿虚线飞回→主调帧右侧落点
落点高亮 400 ms，教师可暂停讲解
```

### 1.5 内存模型图（P0）

#### 1.5.1 对象与引用  

**设计目标**：变量区→对象池用箭头图；实线=强引用，虚线=弱引用；点击对象高亮所有引用，循环引用红色闪电+Tooltip。  

**教学价值**：学生通过点击探索，自主发现「别名」「环」等概念；闪电图标形成「危险」直觉。  

```
内存图示意:
变量区        对象池
a ────▶ [1,2,3]  (引用计数=2)
b ────▶ {"x":9}
点击对象 → 所有引用箭头闪烁
循环引用 → 红色闪电+文字「检测到环」
```

#### 1.5.2 堆-栈-常量分区  

**设计目标**：横向分区色带（栈冰蓝/堆暖橙/常量薄荷绿），地址刻度尺可放大，支持「逻辑视图↔地址视图」切换。  

**教学价值**：色带直接对应「操作系统教材里的内存图」；放大功能满足进阶学生查看「对象头」细节。  

```
分区色带:
栈区 ▊▊▊▊▊▊ 冰蓝
堆区 ▊▊▊▊▊▊ 暖橙
常量区 ▊▊▊▊ 薄荷绿
地址标尺: 相对偏移，可放大
```

### 1.6 控制流高亮（P0）

#### 1.6.1 顺序、分支、循环  

**设计目标**：顺序用淡灰轨迹条；分支被跳过置灰+Tooltip「条件不满足」；循环头部显示「第 i 圈/共 n 圈」徽章+右侧迷你热力条。  

**教学价值**：徽章直接回答「还要循环多少次」；热力条一眼识别热点行。  

```
分支示意:
if x>0:  ▊▊▊▊▊▊ 高亮
    print(x)
else:    ░░░░ 置灰+❌ 条件不满足
循环示意:
for i in range(3): ➊➋➌ 徽章+热力条
```

#### 1.6.2 跳出与返回  

**设计目标**：break/continue/return 用红色「⚡箭头」从关键字射出，指向目标行；动画中途可暂停，箭头停留在「半空」。  

**教学价值**：箭头形成「跳转」直觉；暂停功能让教师有时间追问「跳到了哪里」。  

```
跳出箭头:
break → 移向循环外
return → 弧线射回调用处
```

### 1.7 算法动画模板库（P0）

#### 1.7.1 排序  

**设计目标**：元素=彩色方块，交换=平移+残影；比较=边框闪烁 2 次；完成=整行变绿+滴答提示音；支持「同一数据双算法对比」分屏。  

**教学价值**：方块平移直接对应「交换」概念；滴答声形成「完成」条件反射；对比分屏满足面试「谁更快」追问。  

```
排序动画:
元素=彩色方块，交换=平移+残影
比较=边框闪烁 2 次
完成=整行变绿+滴答提示音
```

#### 1.7.2 搜索  

**设计目标**：二分搜索用「中间高亮→左右遮罩收缩」；命中时目标方块放大+弹跳 1 次；支持自定义数组与目标值。  

**教学价值**：遮罩收缩直接展示「减半」过程；弹跳形成「找到了」惊喜感。  

```
二分搜索:
中间高亮→左右遮罩收缩→命中弹跳
```

#### 1.7.3 链表  

**设计目标**：节点=圆角矩形，next=箭头；插入用「新节点从上方滑入→箭头平滑重连」；支持拖拽节点手动布局。  

**教学价值**：滑入动画形成「插入」直觉；拖拽布局满足「我想自己画一遍」探索需求。  

```
链表插入:
新节点从上方滑入→箭头平滑重连
```

#### 1.7.4 树与图  

**设计目标**：BST 插入用「节点下落→不平衡自动旋转 90°」；图遍历用「节点涂色顺序展示」；最短路径最后高亮闪烁。  

**教学价值**：旋转动画直接对应「左旋/右旋」教材图；涂色顺序形成「遍历」节奏感。  

```
BST 插入:
节点下落→不平衡自动旋转 90°
图遍历:
节点涂色顺序展示，最短路径闪烁
```

#### 1.7.5 动态规划  

**设计目标**：DP 表=网格，当前单元格高亮，依赖单元格用箭头回指；填表完成用「金色轨迹」连成最优路径。  

**教学价值**：箭头回指直接展示「状态转移方程」；金色轨迹形成「最优解」成就感。  

```
DP 表:
当前格高亮+依赖箭头回指
最优路径: 金色轨迹连成线
```

#### 1.7.6 参数即滑即改  

**设计目标**：滑块组（长度/最大值/有序度/重复），释放瞬间动画重播；下方实时显示比较次数、交换次数、耗时曲线。  

**教学价值**：学生通过拖拽即可探索「近乎有序数组对快速排序的影响」；实时曲线形成「实验数据」意识。  

```
滑块组:
长度▶──●──◀最大值▶──●──◀有序度▶──●──◀重复
实时曲线: 比较次数+交换次数+耗时
```

#### 1.7.7 算法对比  

**设计目标**：同一数据，左右分屏，同步步进；差异步骤用红/蓝边框区分；底部实时计数器对比。  

**教学价值**：学生直观看到「快速排序比冒泡少多少次比较」；边框颜色差异形成「谁更快」直觉。  

```
分屏对比:
左：冒泡，右：快速
差异边框红/蓝，底部计数器实时对比
```

### 1.8 面向对象可视化（P0）

#### 1.8.1 类图  

**设计目标**：自动布局（父类在上，子类在下，横向树形）；教师可拖拽保存布局；支持折叠属性与方法区。  

**教学价值**：拖拽保存让教师「一次排版，多次复用」；折叠机制避免「一屏太多字」认知过载。  

```
自动布局:
父类在上，子类在下，横向树形
拖拽后排版记忆到本地
```

#### 1.8.2 实例关系  

**设计目标**：实例=气泡，半径∝属性数量；点击气泡展开详细属性表，并高亮所属类框；同类实例气泡颜色相同。  

**教学价值**：气泡大小直接反映「对象胖瘦」；颜色编码帮助「哪个实例属于哪个类」一眼识别。  

```
实例气泡:
半径∝属性数量，点击展开详情
同类同色，按生成顺序排列
```

#### 1.8.3 多态时序  

**设计目标**：同一消息用彩虹色箭头分别射入不同子类方法；箭头速度相同，营造「同时响应」效果。  

**教学价值**：彩虹色形成「同一接口，不同行为」视觉冲击；速度相同避免「谁先谁后」误解。  

```
彩虹箭头:
同一消息→不同子类方法
箭头速度相同，营造同时响应
```

### 1.9 异常处理可视化（P0）

#### 1.9.1 抛出与传播  

**设计目标**：异常诞生用红色闪电垂直穿透调用栈；每层帧停留 300 ms，显示「未处理」图标；被捕获时闪电被「手套」图标握住，转绿色。  

**教学价值**：闪电形成「异常传播」直觉；手套图标形成「被捕获」安全感。  

```
异常闪电:
红色闪电垂直穿透调用栈
被捕获→转绿色
```

#### 1.9.2 finally 块  

**设计目标**：灰色遮罩覆盖 finally 块，遮罩渐变消失表示执行完成；遮罩期间不允许交互，强化「无论如何都会执行」。  

**教学价值**：遮罩消失形成「终于完成」直觉；不允许交互强化「无法跳过」语义。  

```
finally 遮罩:
灰色覆盖，渐变消失
期间不允许交互，强化必然执行
```

#### 1.9.3 异常分类图标  

**设计目标**：每种异常分配唯一图标+颜色，形成「错误模式库」直觉；图标在错误行、日志、统计三处保持一致。  

**教学价值**：学生看到「颜色」即知「缩进错误」，无需阅读英文异常名；一致性降低记忆负担。  

```
异常图标表:
SyntaxError ⚠️红
IndentationError 📏橙
NameError ❓蓝
TypeError 🔧紫
ValueError 💔深红
```

#### 1.10.1 运算符优先级树  

**设计目标**：树节点=圆角矩形，可折叠；当前求值节点高亮+边框放大；依赖节点用淡蓝箭头回指。  

**教学价值**：折叠机制避免「一整棵树」信息过载；箭头回指直接展示「依赖关系」。  

```
运算符树:
可折叠节点，当前高亮+边框放大
依赖箭头淡蓝回指
```

#### 1.10.2 短路求值  

**设计目标**：and/or 未评估部分置灰+Tooltip「已短路」；评估路径用绿色箭头贯穿「活跃」分支。  

**教学价值**：置灰形成「被跳过」直觉；绿色箭头强化「活跃路径」。  

```
短路示意:
```

### 1.13 教学分享与协作（P2）

#### 1.13.1 链接分享  

**设计目标**：一键生成 8 位随机链接；权限三档：只读👁/可批注✍/可协作🤝；打开链接即自动还原代码、断点、播放位置。  

**教学价值**：教师无需注册账号即可分享；三档权限满足「课后复习→小组讨论→全班协作」递进场景。  

```
链接分享:
权限: 👁只读 ✍批注 🤝协作
```

#### 1.13.2 录屏导出

**设计目标**：叠加层开关：音频波形、字幕、代码行高亮；预览窗口 15 s 可拖动裁剪；导出格式 WebM（含透明通道）+ MP4（通用）。  

**教学价值**：透明通道方便后期合成到 PPT；拖动裁剪让教师快速生成「15 秒精华」微课。  

```
录屏导出:
叠加层: 音频波形+字幕+高亮
预览裁剪 → WebM / MP4
```
